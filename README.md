## Conditional Statement

**If / else**
```
if(condition)
  some statement
  
else if(another condition)
  another statement

else
  yet another statement
```
```
if(condition)
{
   some statements
}

else
{
   other statements
}
```

**Nested if Statement**
```
if(condition)
{
  if(another condition)
  {
    some statements 
  }
  else
  {
    other statements
  }
}
```

**Switch / case**
```
switch(role)
{
  case Role.Admin:
    ...
    break;
    
  case Role.Modertor:
    ...
    break;
    
  default
    ...
    break;
}
```
